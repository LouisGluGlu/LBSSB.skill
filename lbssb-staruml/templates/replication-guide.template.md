# Replication Guide

## Must Prepare

- Source `.mdj`.
- Business input: requirement, screenshot, guide, or use case.
- Output directory.
- StarUML access or fallback renderer.

## Recommended

- Existing use case diagrams.
- Reference screenshots.
- Local `mcp/` and `tools/`.
- `.lbssb/` records.

## First Run

- Create `.lbssb`.
- Read diagram list.
- Extract business objects.
- Produce gap list and DiagramPlan.

## Second Run

- Execute MCP/script plan.
- Export PNG.
- Verify QualityGate.
- Update `next-action.md`.
